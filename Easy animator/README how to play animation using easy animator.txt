How to play animation?

1) Open Easy animator in IntelliJ

2) src ->cs5004.animator -> EasyAnimator

3) Run -> Edit configurations 

4) Fill in command-line arguments:

Example:
-in big-bang-big-crunch.txt -speed 50 -view playback

another example:
-in toh-12.txt -speed 10 -view svg

5) Run (it may take a couple seconds to start)

6) .txt file list:
big-bang-big-crunch.txt
buildings.txt
hanoi.txt
toh-5.txt
toh-8.txt
smalldemo.txt
